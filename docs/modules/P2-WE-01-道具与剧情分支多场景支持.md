# P2-WE-01 道具与剧情节点/分支、多场景支持

## 一、模块概述

- **模块目标**: 实现道具系统、剧情分支系统和多场景切换功能
- **对应 WBS 任务 ID**: P2-WE-01
- **依赖模块**: P1-WE-01, P1-WE-02
- **被依赖模块**: P2-SO-01, P2-UI-01

## 三、API 接口

### 3.1 ItemService

```typescript
class ItemService {
  // 道具定义
  registerItem(item: Item): void;
  getItem(itemId: string): Item | undefined;
  getAllItems(): Item[];

  // 道具实例
  giveItem(
    worldState: WorldState,
    itemId: string,
    ownerId: string,
    quantity?: number
  ): string;
  removeItem(worldState: WorldState, instanceId: string): boolean;
  getOwnerItems(
    worldState: WorldState,
    ownerId: string
  ): Array<{
    instanceId: string;
    instance: ItemInstance;
    item: Item | undefined;
  }>;
  transferItem(
    worldState: WorldState,
    instanceId: string,
    newOwnerId: string
  ): boolean;
}
```

### 3.2 PlotService

```typescript
class PlotService {
  // 剧情图管理
  loadPlotGraph(config: PlotGraphConfig): void;
  getNode(nodeId: string): PlotNode | undefined;
  getStartNodeId(): string | undefined;

  // 分支管理
  getAvailableBranches(worldState: WorldState): PlotBranch[];
  advanceBranch(
    worldState: WorldState,
    branchId: string
  ): PlotBranch | undefined;
  setCurrentNode(worldState: WorldState, nodeId: string): void;
  isTerminal(worldState: WorldState): boolean;
}
```

### 3.3 SceneManager

```typescript
class SceneManager {
  // 场景管理
  registerScene(scene: Scene): void;
  registerScenes(scenes: Scene[]): void;
  getScene(sceneId: string): Scene | undefined;
  getAllScenes(): Scene[];

  // 场景切换
  registerTransition(transition: SceneTransition): void;
  getAvailableTransitions(sceneId: string): SceneTransition[];
  canTransition(
    worldState: WorldState,
    transitionId: string,
    characterIds: string[]
  ): {
    canTransition: boolean;
    reason?: string;
  };
  executeTransition(
    worldState: WorldState,
    transitionId: string,
    characterIds: string[],
    reason?: string
  ): {
    success: boolean;
    error?: string;
  };
  changeScene(
    worldState: WorldState,
    toSceneId: string,
    characterIds: string[],
    reason?: string
  ): {
    success: boolean;
    error?: string;
  };

  // 场景角色管理
  getCharactersInScene(sceneId: string): string[];
  addCharacterToScene(sceneId: string, characterId: string): boolean;
  removeCharacterFromScene(sceneId: string, characterId: string): boolean;

  // 历史记录
  getTransitionHistory(limit?: number): SceneTransitionRecord[];
}
```

## 七、已知限制

1. **剧情分支条件**: 当前 PlotService 的分支条件检查较简单，未实现复杂条件判断
2. **场景容量**: 场景中的角色和道具数量暂无上限限制
3. **切换动画**: 场景切换暂不支持动画效果配置
4. **场景状态**: 场景的动态状态（如天气、时间）暂未实现
5. **道具效果**: 道具的使用效果需要外部实现，ItemService 仅管理道具实例
