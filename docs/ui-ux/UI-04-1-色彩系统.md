## 一、Material Design 3 色彩系统

### 1.1 色彩角色（Color Roles）

Material Design 3 使用基于角色的色彩系统，每种颜色都有明确的用途。

| 角色 | 说明 | 使用场景 |
|------|------|----------|
| Primary | 主色 | 主要按钮、FAB、高亮元素 |
| Secondary | 辅助色 | 次要按钮、辅助元素 |
| Tertiary | 第三色 | 强调、对比元素 |
| Error | 错误色 | 错误提示、警告 |
| Surface | 表面色 | 卡片、对话框背景 |
| Background | 背景色 | 页面背景 |
| Outline | 轮廓色 | 边框、分隔线 |

## 四、色彩使用指南

### 4.1 色彩对比度

**WCAG 2.1 标准**
- **AA 级（最低要求）**：
  - 正文文字：4.5:1
  - 大文字（18sp+ 或 14sp+ 粗体）：3:1
  - 交互元素：3:1
- **AAA 级（增强）**：
  - 正文文字：7:1
  - 大文字：4.5:1

**本项目要求**
- 正文文字：≥ 4.5:1（AA 级）
- 标题文字：≥ 7:1（AAA 级）
- 交互元素：≥ 3:1（AA 级）

### 4.2 色彩应用场景

| 场景 | 浅色模式 | 深色模式 | 说明 |
|------|----------|----------|------|
| 主要按钮 | primary | primary | 最重要的操作 |
| 次要按钮 | secondary | secondary | 次要操作 |
| 文字按钮 | on-surface | on-surface | 低优先级操作 |
| 链接 | primary | primary | 可点击链接 |
| 成功提示 | success | success | 操作成功 |
| 警告提示 | warning | warning | 需要注意 |
| 错误提示 | error | error | 操作失败 |
| 信息提示 | info | info | 一般信息 |
| 禁用状态 | grey-400 | grey-600 | 不可交互 |

### 4.3 色彩组合示例

**主要操作**
- 背景：primary
- 文字：on-primary
- 悬停：primary-darken-1
- 按下：primary-darken-2
- 禁用：grey-400

**次要操作**
- 背景：secondary
- 文字：on-secondary
- 悬停：secondary-darken-1
- 按下：secondary-darken-2

**成功状态**
- 背景：success-container
- 文字：on-success-container
- 图标：success

**错误状态**
- 背景：error-container
- 文字：on-error-container
- 图标：error

## 五、色彩实现

### 5.1 Vuetify 3 色彩配置

```typescript
// vuetify.config.ts
import { createVuetify } from 'vuetify'

export default createVuetify({
  theme: {
    defaultTheme: 'light',
    themes: {
      light: {
        colors: {
          primary: '#1976D2',
          secondary: '#FF6F00',
          tertiary: '#00897B',
          success: '#4CAF50',
          warning: '#FF9800',
          error: '#F44336',
          info: '#2196F3',
          surface: '#FFFFFF',
          background: '#FAFAFA',
        },
      },
      dark: {
        colors: {
          primary: '#90CAF9',
          secondary: '#FFB74D',
          tertiary: '#80CBC4',
          success: '#81C784',
          warning: '#FFB74D',
          error: '#E57373',
          info: '#64B5F6',
          surface: '#121212',
          background: '#0A0A0A',
        },
      },
    },
  },
})
```

```css
/* 浅色模式 */
```
