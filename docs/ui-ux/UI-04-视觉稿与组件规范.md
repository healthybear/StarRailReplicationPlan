## 一、设计系统概览

### 1.1 Material Design 3 核心原则

1. **个性化（Personalization）**：支持动态主题色，适应用户偏好
2. **无障碍（Accessibility）**：确保所有用户都能使用
3. **自适应（Adaptive）**：响应不同设备和屏幕尺寸
4. **表现力（Expressive）**：现代化、有活力的视觉语言

### 1.2 设计 Token 体系

| Token 类型 | 说明 | 示例 |
|-----------|------|------|
| 色彩 Token | 主色、辅助色、语义色 | primary, secondary, error |
| 排版 Token | 字体、字号、行高 | headline-large, body-medium |
| 间距 Token | 内边距、外边距 | spacing-sm, spacing-lg |
| 圆角 Token | 组件圆角 | rounded-sm, rounded-lg |
| 阴影 Token | 层级阴影 | elevation-1, elevation-4 |

## 三、设计工作流

### 3.1 设计工具设置

**Figma 设置**
1. 安装 Material Design 3 插件
2. 导入 Vuetify 3 组件库
3. 设置设计 Token（色彩、排版、间距）
4. 创建组件库（Components）
5. 设置响应式断点（xs/sm/md/lg/xl）

**Sketch 设置**
1. 安装 Material Design 插件
2. 导入 Vuetify 3 Sketch 资源
3. 创建 Symbols 库
4. 设置 Text Styles 和 Layer Styles

### 3.2 设计交付物

| 交付物 | 格式 | 说明 |
|--------|------|------|
| 线框稿 | Figma/Sketch | 低保真，展示布局和信息架构 |
| 视觉稿 | Figma/Sketch | 高保真，完整视觉效果 |
| 切图资源 | SVG/PNG | 图标、插图等静态资源 |
| 设计规范 | PDF/Figma | 色彩、排版、组件规范文档 |
| 原型 | Figma Prototype | 可交互原型，展示交互流程 |

### 3.3 设计评审检查清单

**视觉一致性**
- [ ] 色彩使用符合设计系统
- [ ] 排版层级清晰
- [ ] 间距统一（4dp 倍数）
- [ ] 圆角统一
- [ ] 阴影层级正确

**响应式设计**
- [ ] 覆盖所有断点（xs/sm/md/lg/xl）
- [ ] 移动端触控区域 ≥ 48dp
- [ ] 文字可读性良好
- [ ] 图片自适应

**无障碍**
- [ ] 颜色对比度 ≥ 4.5:1
- [ ] 交互元素对比度 ≥ 3:1
- [ ] 不仅依赖颜色传达信息
- [ ] 焦点状态清晰可见

**性能**
- [ ] 图片优化（WebP 格式）
- [ ] 图标使用 SVG
- [ ] 避免过度动效
- [ ] 字体文件优化

## 四、深色模式设计指南

### 4.1 深色模式原则

1. **降低亮度**：背景使用深灰色（#121212）而非纯黑
2. **提升对比**：文字和图标使用浅色
3. **减少饱和度**：色彩饱和度降低 10-20%
4. **保持层级**：使用阴影和表面色区分层级

### 4.2 深色模式切换

- **自动切换**：跟随系统设置
- **手动切换**：提供切换按钮（设置页面）
- **记忆偏好**：保存用户选择到本地存储

### 4.3 深色模式测试

- [ ] 所有页面在深色模式下可读
- [ ] 色彩对比度符合标准
- [ ] 图片和图标适配深色背景
- [ ] 动效在深色模式下流畅

## 五、图标使用规范

### 5.1 图标库

**Material Design Icons**
- 官网：https://materialdesignicons.com/
- 使用方式：通过 Vuetify 的 `mdi-` 前缀引用
- 示例：`mdi-account`, `mdi-home`, `mdi-settings`

### 5.2 图标尺寸

| 尺寸 | 值 | 用途 |
|------|-----|------|
| 小 | 16dp | 列表项、标签 |
| 中 | 20dp | 按钮、输入框 |
| 大 | 24dp | 导航栏、工具栏 |
| 超大 | 48dp | 空状态、引导页 |

### 5.3 图标颜色

- **默认**：继承父元素文字颜色
- **主色**：`color="primary"`
- **语义色**：`color="success/warning/error"`
- **自定义**：使用 CSS 变量

## 六、动效设计原则

### 6.1 Material Motion

Material Design 3 动效遵循以下原则：
1. **有目的**：动效应帮助用户理解界面变化
2. **快速**：动效时长通常 200-400ms
3. **自然**：使用缓动函数模拟物理运动
4. **一致**：相同类型的动效使用相同参数

### 6.2 缓动函数

| 函数 | 用途 | 示例 |
|------|------|------|
| ease-in-out | 通用过渡 | 页面切换、对话框 |
| ease-out | 元素进入 | 淡入、滑入 |
| ease-in | 元素退出 | 淡出、滑出 |
| linear | 持续动画 | 加载动画、进度条 |

### 6.3 动效时长

| 时长 | 用途 |
|------|------|
| 100ms | 微交互（按钮点击） |
| 200ms | 快速过渡（颜色变化） |
| 300ms | 标准过渡（页面切换） |
| 500ms | 复杂动画（数值滚动） |
| 1000ms | 特殊效果（加载动画） |
