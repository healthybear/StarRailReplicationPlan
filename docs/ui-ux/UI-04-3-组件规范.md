## 文档说明

## 一、按钮（v-btn）

### 1.1 组件说明
按钮用于触发操作，是最常用的交互元素。

### 1.2 变体（Variant）

| 变体 | 说明 | 用途 |
|------|------|------|
| elevated | 凸起按钮，有阴影 | 主要操作 |
| filled | 填充按钮，无阴影 | 次要操作 |
| tonal | 色调按钮，浅色背景 | 辅助操作 |
| outlined | 轮廓按钮，仅边框 | 低优先级操作 |
| text | 文字按钮，无背景 | 最低优先级操作 |
| plain | 纯文字，无任何装饰 | 链接样式 |

### 1.3 尺寸（Size）

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| x-small | 24dp | 4dp 8dp | 12sp | 极小按钮 |
| small | 32dp | 6dp 12dp | 13sp | 小按钮 |
| default | 40dp | 8dp 16dp | 14sp | 标准按钮 |
| large | 48dp | 10dp 20dp | 15sp | 大按钮 |
| x-large | 56dp | 12dp 24dp | 16sp | 超大按钮 |

### 1.4 状态

**默认状态**
- 背景：primary
- 文字：on-primary
- 圆角：4dp
- 阴影：elevation-2（elevated 变体）

**悬停状态（Hover）**
- 背景：primary-darken-1
- 叠加层：8% 白色
- 阴影：elevation-4（elevated 变体）
- 过渡：200ms

**按下状态（Active）**
- 背景：primary-darken-2
- 叠加层：12% 白色
- 阴影：elevation-8（elevated 变体）

**焦点状态（Focus）**
- 轮廓：2dp primary 色
- 轮廓偏移：2dp

**禁用状态（Disabled）**
- 背景：grey-300（浅色模式）、grey-700（深色模式）
- 文字：grey-500
- 不透明度：38%
- 无阴影

### 1.5 图标按钮

**尺寸**
- 最小点击区域：48dp × 48dp
- 图标尺寸：24dp
- 内边距：12dp

**变体**
- icon：圆形图标按钮
- fab：浮动操作按钮（56dp × 56dp）
- fab-small：小型 FAB（40dp × 40dp）

## 二、卡片（v-card）

### 2.1 组件说明
卡片用于展示内容和操作，是信息的容器。

### 2.2 变体

| 变体 | 说明 | 用途 |
|------|------|------|
| elevated | 凸起卡片，有阴影 | 默认样式 |
| flat | 平面卡片，无阴影 | 列表项 |
| tonal | 色调卡片，浅色背景 | 强调内容 |
| outlined | 轮廓卡片，仅边框 | 次要内容 |

### 2.3 结构

```
v-card
├─ v-card-title (标题区)
│  ├─ 字号：22sp (title-large)
│  ├─ 字重：500
│  └─ 内边距：16dp
├─ v-card-subtitle (副标题区)
│  ├─ 字号：14sp (body-medium)
│  ├─ 颜色：medium-emphasis
│  └─ 内边距：0 16dp 16dp
├─ v-card-text (内容区)
│  ├─ 字号：16sp (body-large)
│  └─ 内边距：16dp
└─ v-card-actions (操作区)
   ├─ 对齐：右对齐
   ├─ 内边距：8dp 16dp
   └─ 按钮间距：8dp
```

### 2.4 尺寸

- 最小宽度：280dp
- 最大宽度：无限制（响应式）
- 内边距：16dp
- 圆角：8dp
- 阴影：elevation-2（elevated 变体）

### 2.5 状态

**默认状态**
- 背景：surface
- 阴影：elevation-2

**悬停状态**
- 阴影：elevation-4
- 过渡：300ms

**按下状态**
- 阴影：elevation-8

## 三、输入框（v-text-field）

### 3.1 组件说明
输入框用于接收用户文本输入。

### 3.2 变体

| 变体 | 说明 | 用途 |
|------|------|------|
| filled | 填充输入框 | 默认样式 |
| outlined | 轮廓输入框 | 强调边界 |
| plain | 纯文本输入框 | 简洁样式 |
| underlined | 下划线输入框 | 紧凑样式 |

### 3.3 结构

```
v-text-field
├─ label (标签)
│  ├─ 字号：16sp → 12sp (聚焦时)
│  ├─ 位置：输入框内 → 输入框上方
│  └─ 颜色：medium-emphasis → primary
├─ input (输入区)
│  ├─ 字号：16sp
│  ├─ 高度：56dp
│  └─ 内边距：16dp
├─ prepend-icon (前置图标)
│  ├─ 尺寸：24dp
│  └─ 间距：12dp
├─ append-icon (后置图标)
│  ├─ 尺寸：24dp
│  └─ 间距：12dp
└─ messages (提示/错误信息)
   ├─ 字号：12sp
   ├─ 颜色：medium-emphasis / error
   └─ 间距：4dp
```

### 3.4 状态

**默认状态**
- 背景：surface-variant
- 边框：1dp outline
- 圆角：4dp

**悬停状态**
- 背景：surface-variant（叠加 4% on-surface）
- 边框：1dp on-surface

**聚焦状态**
- 边框：2dp primary
- 标签颜色：primary
- 标签上移并缩小

**错误状态**
- 边框：2dp error
- 标签颜色：error
- 错误信息：error 色

**禁用状态**
- 背景：surface-variant（38% 不透明度）
- 文字：disabled
- 不可交互

## 四、多行输入（v-textarea）

### 4.1 组件说明
多行输入框用于接收多行文本输入。

### 4.2 特性

- 自动增长：`auto-grow` 属性
- 最小行数：`rows` 属性（默认 5）
- 最大行数：`max-rows` 属性
- 字符计数：`counter` 属性

### 4.3 尺寸

- 最小高度：80dp（2 行）
- 最大高度：200dp（可配置）
- 内边距：16dp
- 行高：24sp

---

## 五、下拉选择（v-select）

### 5.1 组件说明
下拉选择用于从预定义选项中选择。

### 5.2 结构

```
v-select
├─ 输入区（同 v-text-field）
├─ 下拉图标（mdi-menu-down）
└─ 下拉菜单（v-menu）
   ├─ 最大高度：304dp（约 8 项）
   ├─ 阴影：elevation-8
   └─ v-list
      └─ v-list-item × N
         ├─ 高度：48dp
         ├─ 内边距：16dp
         └─ 选中状态：primary 背景（8% 不透明度）
```

### 5.3 状态

**选中项**
- 背景：primary（8% 不透明度）
- 文字：primary
- 勾选图标：mdi-check

**悬停项**
- 背景：on-surface（4% 不透明度）

---

## 六、对话框（v-dialog）

### 6.1 组件说明
对话框用于显示需要用户交互的内容。

### 6.2 结构

```
v-dialog
├─ 遮罩层（scrim）
│  ├─ 背景：black
│  ├─ 不透明度：32%
│  └─ 点击关闭：可配置
└─ v-card（对话框内容）
   ├─ 最大宽度：600dp（可配置）
   ├─ 圆角：12dp
   ├─ 阴影：elevation-24
   └─ 内边距：24dp
```

### 6.3 尺寸

| 类型 | 最大宽度 | 用途 |
|------|----------|------|
| 小 | 400dp | 确认对话框 |
| 中 | 600dp | 表单对话框 |
| 大 | 800dp | 内容对话框 |
| 全屏 | 100% | 移动端 |

### 6.4 动画

- 进入：缩放 + 淡入，300ms
- 退出：缩放 + 淡出，200ms
- 缓动：ease-out

---

## 七、列表（v-list）

### 7.1 组件说明
列表用于展示一组相关项目。

### 7.2 列表项结构

```
v-list-item
├─ prepend（前置内容）
│  ├─ v-avatar（头像，40dp）
│  └─ v-icon（图标，24dp）
├─ 内容区
│  ├─ v-list-item-title（标题，16sp）
│  ├─ v-list-item-subtitle（副标题，14sp）
│  └─ v-list-item-text（描述，14sp）
└─ append（后置内容）
   ├─ v-icon（图标，24dp）
   └─ v-chip（标签）
```

### 7.3 列表项高度

| 类型 | 高度 | 用途 |
|------|------|------|
| 单行 | 48dp | 仅标题 |
| 双行 | 64dp | 标题 + 副标题 |
| 三行 | 88dp | 标题 + 副标题 + 描述 |

### 7.4 状态

**默认状态**
- 背景：transparent
- 内边距：0 16dp

**悬停状态**
- 背景：on-surface（4% 不透明度）

**选中状态**
- 背景：primary（8% 不透明度）
- 文字：primary

**激活状态**
- 背景：primary（12% 不透明度）

---

## 八、标签（v-chip）

### 8.1 组件说明
标签用于展示标签、状态或过滤器。

### 8.2 尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| x-small | 20dp | 4dp 8dp | 10sp | 极小标签 |
| small | 24dp | 6dp 12dp | 12sp | 小标签 |
| default | 32dp | 8dp 12dp | 14sp | 标准标签 |
| large | 40dp | 10dp 16dp | 16sp | 大标签 |

### 8.3 变体

| 变体 | 说明 | 用途 |
|------|------|------|
| elevated | 凸起标签 | 强调 |
| flat | 平面标签 | 默认 |
| tonal | 色调标签 | 状态 |
| outlined | 轮廓标签 | 过滤器 |

### 8.4 特性

- 可关闭：`closable` 属性，显示关闭图标
- 可点击：`clickable` 属性，悬停效果
- 圆角：16dp（半圆形）

---

## 九、头像（v-avatar）

### 9.1 组件说明
头像用于展示用户或实体的图像。

### 9.2 尺寸

| 尺寸 | 直径 | 用途 |
|------|------|------|
| x-small | 24dp | 列表项、标签 |
| small | 32dp | 小头像 |
| default | 40dp | 标准头像 |
| large | 56dp | 大头像 |
| x-large | 72dp | 个人资料 |

### 9.3 变体

- 图片头像：显示图片
- 文字头像：显示首字母
- 图标头像：显示图标

### 9.4 样式

- 形状：圆形（默认）、方形（`rounded="0"`）
- 边框：可选，2dp
- 背景：primary（文字头像）

---

## 十、进度条（v-progress-linear）

### 10.1 组件说明
进度条用于展示操作进度。

### 10.2 尺寸

- 高度：4dp（默认）、8dp（强调）
- 宽度：100%（父容器）
- 圆角：2dp

### 10.3 模式

| 模式 | 说明 | 用途 |
|------|------|------|
| determinate | 确定进度 | 文件上传 |
| indeterminate | 不确定进度 | 加载中 |
| buffer | 缓冲进度 | 视频加载 |
| query | 查询进度 | 数据查询 |

### 10.4 颜色

- 进度条：primary
- 背景：primary（12% 不透明度）
- 缓冲区：primary（24% 不透明度）

---

## 十一、提示条（v-snackbar）

### 11.1 组件说明
提示条用于显示简短的消息。

### 11.2 结构

```
v-snackbar
├─ 位置：底部居中（默认）
├─ 最大宽度：672dp
├─ 最小宽度：344dp
├─ 高度：48dp（单行）、68dp（双行）
├─ 内边距：14dp 16dp
├─ 圆角：4dp
└─ 阴影：elevation-6
```

### 11.3 位置

- top：顶部居中
- bottom：底部居中（默认）
- left：左侧
- right：右侧

### 11.4 特性

- 自动关闭：`timeout` 属性（默认 5000ms）
- 操作按钮：可选，右侧
- 关闭按钮：可选，右侧图标

---

## 十二、警告框（v-alert）

### 12.1 组件说明
警告框用于显示重要信息。

### 12.2 类型

| 类型 | 颜色 | 图标 | 用途 |
|------|------|------|------|
| success | success | mdi-check-circle | 成功提示 |
| info | info | mdi-information | 信息提示 |
| warning | warning | mdi-alert | 警告提示 |
| error | error | mdi-alert-circle | 错误提示 |

### 12.3 变体

| 变体 | 说明 | 用途 |
|------|------|------|
| elevated | 凸起警告框 | 强调 |
| flat | 平面警告框 | 默认 |
| tonal | 色调警告框 | 柔和 |
| outlined | 轮廓警告框 | 次要 |

### 12.4 结构

```
v-alert
├─ 图标（左侧，24dp）
├─ 标题（可选，16sp）
├─ 内容（14sp）
├─ 关闭按钮（可选，右侧）
├─ 内边距：16dp
└─ 圆角：4dp
```
