## 一、排版系统

### 1.1 字体家族

**主字体：Roboto**
- 用途：界面文字、正文、标题
- 字重：300 (Light), 400 (Regular), 500 (Medium), 700 (Bold)
- 来源：Google Fonts
- 备用字体：-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

**等宽字体：Roboto Mono**
- 用途：代码、数据展示
- 字重：400 (Regular), 700 (Bold)

**中文字体**
- 主字体：思源黑体（Noto Sans SC）
- 备用：苹方（PingFang SC）、微软雅黑（Microsoft YaHei）

### 1.2 字体加载

```html
<!-- Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500;700&family=Roboto+Mono:wght@400;700&display=swap" rel="stylesheet">
```

## 二、文本样式规范

### 2.1 Display（超大标题）

| 样式 | 字号 | 行高 | 字重 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Display Large | 57sp | 64sp | 400 | -0.25sp | 营销页面、引导页 |
| Display Medium | 45sp | 52sp | 400 | 0sp | 大型标题 |
| Display Small | 36sp | 44sp | 400 | 0sp | 页面主标题 |

**使用场景**：
- 引导页标题
- 空状态标题
- 营销页面标题

**示例**：
```html
<h1 class="text-display-large">星穹铁道剧情复现计划</h1>
```

### 2.2 Headline（标题）

| 样式 | 字号 | 行高 | 字重 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Headline Large | 32sp | 40sp | 400 | 0sp | 页面标题 |
| Headline Medium | 28sp | 36sp | 400 | 0sp | 区块标题 |
| Headline Small | 24sp | 32sp | 400 | 0sp | 卡片标题 |

**使用场景**：
- 页面标题（H1）
- 对话框标题
- 区块标题（H2）

**示例**：
```html
<h1 class="text-headline-large">会话列表</h1>
<h2 class="text-headline-medium">最近会话</h2>
```

### 2.3 Title（小标题）

| 样式 | 字号 | 行高 | 字重 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Title Large | 22sp | 28sp | 500 | 0sp | 卡片标题 |
| Title Medium | 16sp | 24sp | 500 | 0.15sp | 列表项标题 |
| Title Small | 14sp | 20sp | 500 | 0.1sp | 小组件标题 |

**使用场景**：
- 卡片标题
- 列表项标题
- 标签页标题

**示例**：
```html
<v-card-title class="text-title-large">会话名称</v-card-title>
<v-list-item-title class="text-title-medium">人物名称</v-list-item-title>
```

### 2.4 Body（正文）

| 样式 | 字号 | 行高 | 字重 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Body Large | 16sp | 24sp | 400 | 0.5sp | 主要正文 |
| Body Medium | 14sp | 20sp | 400 | 0.25sp | 次要正文 |
| Body Small | 12sp | 16sp | 400 | 0.4sp | 辅助文字 |

**使用场景**：
- 正文内容
- 对话内容
- 描述文字

**示例**：
```html
<p class="text-body-large">这是主要正文内容</p>
<p class="text-body-medium">这是次要正文内容</p>
```

### 2.5 Label（标签）

| 样式 | 字号 | 行高 | 字重 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| Label Large | 14sp | 20sp | 500 | 0.1sp | 按钮文字 |
| Label Medium | 12sp | 16sp | 500 | 0.5sp | 标签、徽章 |
| Label Small | 11sp | 16sp | 500 | 0.5sp | 小标签 |

**使用场景**：
- 按钮文字
- 标签（v-chip）
- 表单标签

**示例**：
```html
<v-btn class="text-label-large">确认</v-btn>
<v-chip class="text-label-medium">标签</v-chip>
```

## 三、排版层级

### 3.1 层级关系

```
Display Large (57sp)
  └─ Display Medium (45sp)
      └─ Display Small (36sp)
          └─ Headline Large (32sp)
              └─ Headline Medium (28sp)
                  └─ Headline Small (24sp)
                      └─ Title Large (22sp)
                          └─ Body Large (16sp)
                              └─ Title Medium (16sp)
                                  └─ Body Medium (14sp)
                                      └─ Title Small (14sp)
                                          └─ Label Large (14sp)
                                              └─ Body Small (12sp)
                                                  └─ Label Medium (12sp)
                                                      └─ Label Small (11sp)
```

### 3.2 页面排版示例

**会话列表页**
```
Headline Large (32sp) - "会话列表"

```

**剧情推进页**
```
Headline Medium (28sp) - 场景描述标题

  ├─ Title Medium (16sp) - 人物名称
  ├─ Body Large (16sp) - 对话内容
  └─ Body Small (12sp) - 时间戳
```

## 四、间距系统

### 4.1 间距 Token

| Token | 值 | 用途 |
|-------|-----|------|
| spacing-xs | 4dp | 最小间距，图标与文字 |
| spacing-sm | 8dp | 小间距，列表项内部 |
| spacing-md | 16dp | 标准间距，卡片内边距 |
| spacing-lg | 24dp | 大间距，区块间距 |
| spacing-xl | 32dp | 超大间距，页面边距 |
| spacing-xxl | 48dp | 特大间距，区块分隔 |
| spacing-xxxl | 64dp | 巨大间距，页面顶部 |

### 4.2 间距使用规则

**4dp 基础单位**
- 所有间距必须是 4dp 的倍数
- 示例：4dp, 8dp, 12dp, 16dp, 20dp, 24dp...

**常用间距值**
- 4dp：图标与文字、标签内边距
- 8dp：列表项内部、按钮内边距
- 12dp：小卡片内边距
- 16dp：标准卡片内边距、列表项间距
- 24dp：区块间距、容器内边距
- 32dp：页面边距（桌面端）
- 48dp：大区块分隔
- 64dp：页面顶部间距

### 4.3 组件间距规范

**卡片（v-card）**
- 内边距：16dp
- 标题与内容间距：12dp
- 内容与操作按钮间距：16dp
- 卡片间距：16dp

**列表（v-list）**
- 列表项高度：48dp（单行）、64dp（双行）、88dp（三行）
- 列表项内边距：16dp
- 列表项间距：0dp（无分隔线）、1dp（有分隔线）
- 图标与文字间距：16dp

**按钮（v-btn）**
- 内边距：8dp 16dp（垂直 水平）
- 按钮间距：8dp
- 图标与文字间距：8dp

**输入框（v-text-field）**
- 高度：56dp
- 内边距：16dp
- 标签与输入框间距：4dp
- 输入框间距：16dp

**对话框（v-dialog）**
- 内边距：24dp
- 标题与内容间距：16dp
- 内容与按钮间距：24dp
- 按钮间距：8dp

## 五、响应式排版

### 5.1 断点调整

| 断点 | 字号调整 | 间距调整 | 说明 |
|------|----------|----------|------|
| xs (<600px) | -2sp | -4dp | 手机端，减小字号和间距 |
| sm (600-960px) | -1sp | -2dp | 平板竖屏，略微减小 |
| md (960-1280px) | 0sp | 0dp | 平板横屏，标准尺寸 |
| lg (1280-1920px) | 0sp | 0dp | 桌面端，标准尺寸 |
| xl (>1920px) | +2sp | +4dp | 大屏，增大字号和间距 |

### 5.2 移动端优化

**字号调整**
- Display Large: 57sp → 45sp
- Headline Large: 32sp → 28sp
- Body Large: 16sp → 14sp

**间距调整**
- 页面边距：32dp → 16dp
- 卡片内边距：16dp → 12dp
- 区块间距：24dp → 16dp

**行高调整**
- 移动端行高增加 10%，提升可读性
- Body Large: 24sp → 26sp
- Body Medium: 20sp → 22sp

## 六、文本处理

### 6.1 文本截断

**单行截断**
```css
.text-truncate {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

**多行截断（2 行）**
```css
.text-truncate-2 {
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
```

### 6.2 文本对齐

| 对齐方式 | CSS 类 | 用途 |
|----------|--------|------|
| 左对齐 | text-left | 默认，正文 |
| 居中对齐 | text-center | 标题、空状态 |
| 右对齐 | text-right | 数字、时间 |
| 两端对齐 | text-justify | 长文本 |

### 6.3 文本颜色

| 颜色 | CSS 类 | 用途 |
|------|--------|------|
| 主要文字 | text-high-emphasis | 标题、正文 |
| 次要文字 | text-medium-emphasis | 描述、辅助文字 |
| 禁用文字 | text-disabled | 禁用状态 |
| 主色文字 | text-primary | 链接、强调 |
| 错误文字 | text-error | 错误提示 |

**透明度**
- 主要文字：87% 不透明度
- 次要文字：60% 不透明度
- 禁用文字：38% 不透明度

## 七、排版实现

### 7.1 Vuetify 3 排版类

```html
<!-- Display -->
<h1 class="text-h1">Display Large</h1>
<h2 class="text-h2">Display Medium</h2>
<h3 class="text-h3">Display Small</h3>

<!-- Headline -->
<h4 class="text-h4">Headline Large</h4>
<h5 class="text-h5">Headline Medium</h5>
<h6 class="text-h6">Headline Small</h6>

<!-- Body -->
<p class="text-body-1">Body Large</p>
<p class="text-body-2">Body Medium</p>

<!-- Label -->
<span class="text-button">Label Large</span>
<span class="text-caption">Label Medium</span>
<span class="text-overline">Label Small</span>
```

### 7.2 CSS 变量

```css
:root {
  /* 字号 */
  --text-display-large: 57px;
  --text-headline-large: 32px;
  --text-title-large: 22px;
  --text-body-large: 16px;
  --text-label-large: 14px;

  /* 行高 */
  --line-height-display-large: 64px;
  --line-height-headline-large: 40px;
  --line-height-title-large: 28px;
  --line-height-body-large: 24px;
  --line-height-label-large: 20px;

  /* 间距 */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --spacing-xl: 32px;
  --spacing-xxl: 48px;
  --spacing-xxxl: 64px;
}
```

### 7.3 自定义排版类

```css
/* 文本截断 */
```

## 八、排版检查清单

### 8.1 可读性检查

- [ ] 正文字号 ≥ 14sp（移动端）、16sp（桌面端）
- [ ] 行高 ≥ 1.5 倍字号
- [ ] 行长度 ≤ 75 字符（中文 ≤ 40 字符）
- [ ] 段落间距 ≥ 1 倍行高
- [ ] 文字颜色对比度 ≥ 4.5:1

### 8.2 层级检查

- [ ] 标题层级清晰（H1 > H2 > H3）
- [ ] 字号差异明显（至少 2sp）
- [ ] 字重区分合理
- [ ] 颜色强调适当

### 8.3 间距检查

- [ ] 所有间距是 4dp 的倍数
- [ ] 相关元素间距小，无关元素间距大
- [ ] 页面边距统一
- [ ] 移动端间距适当减小
