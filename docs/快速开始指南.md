# 快速开始指南

欢迎使用《星穹铁道剧情复现计划》！本指南将帮助你快速上手项目。

## 📋 前置要求

### 必需软件

- **Node.js** >= 20.0.0 ([下载地址](https://nodejs.org/))
- **pnpm** >= 8.0.0 (安装命令: `npm install -g pnpm`)
- **Git** (用于克隆项目)

### LLM API Key

你需要至少一个 LLM API Key：

- **Deepseek API Key** (推荐) - [获取地址](https://platform.deepseek.com/)
- **Claude API Key** (可选) - [获取地址](https://console.anthropic.com/)

## 🚀 安装步骤

### 1. 克隆项目

\`\`\`bash
git clone <repository-url>
cd StarRailReplicationPlan
\`\`\`

### 2. 安装依赖

\`\`\`bash
pnpm install
\`\`\`

### 3. 配置环境变量

\`\`\`bash
cp .env.example .env

# 编辑 .env 文件，填入你的 API Key

\`\`\`

### 4. 构建项目

\`\`\`bash
pnpm build
\`\`\`

## 🎮 使用方式

### 方式 1: Web 界面（推荐）

\`\`\`bash

# 终端 1: 启动后端

cd packages/api && pnpm start:dev

# 终端 2: 启动前端

cd packages/web && pnpm dev
\`\`\`

访问: http://localhost:5173

### 方式 2: CLI 交互界面

\`\`\`bash
pnpm start
\`\`\`

### 方式 3: REST API

启动 API 服务器后访问: http://localhost:3000/api/docs

详见 [API 使用指南](./API使用指南.md)
